Differences Between Old and New Versions of HTML & CSS

HTML and CSS have undergone significant transformations since their inception.

HTML

HTML (Hyper Text Markup Language) is the standard markup language for creating Web pages. HTML describes the structure of a Web page through a series of elements that tell the browser how to display the content.

Evolution:

  • HTML4: Introduced in 1997, focused on structure over presentation. It provided basic structural elements but required additional technologies like CSS for styling.
  • XHTML: A stricter, XML-based version of HTML 4, introduced in 2000. It enforced stricter syntax rules, making it more predictable but also more verbose.
  • HTML5: Introduced in 2014, brought new semantic elements (like header, footer, and article), multimedia support (like audio and video), and powerful APIs (like Canvas, Web Workers, and Web Storage).

New HTML5 Tags:

  • header: Represents introductory content, typically a group of introductory or navigational aids.
  • nav: Defines a set of navigation links.
  • article: Represents a self-contained composition in a document, page, application, or site, intended to be independently distributable or reusable.
  • section: Represents a standalone section which doesn't have a more specific semantic element to represent it.
  • aside: Represents a portion of a document whose content is only indirectly related to the document's main content.
  • footer: Represents a footer for its nearest sectioning content or sectioning root element.
  • figure: Represents self-contained content, typically with a caption (figcaption), and is typically referenced as a single unit.
  • figcaption: Represents a caption or legend for the content of its parent
  • mark: Represents text which is marked or highlighted for reference purposes.

CSS

CSS (Cascading Style Sheets) is a language used for describing the presentation of a document written in HTML or XML. CSS is essential for designing and structuring web pages, and it has evolved significantly over the years.

Evolution:

  • CSS1: Introduced in 1996, it provided basic style rules, allowing for separation of content and design. It included properties for fonts, colors, and margins.
  • CSS2: Released in 1998, added more capabilities like positioning, media types, and a wider range of selectors. It allowed for more complex designs and responsive layouts.
  • CSS3: Introduced a modular design, making it easier to update and maintain. CSS3 brought animations, transitions, flexbox, grid layout, and many other advanced features for sophisticated web designs.

New CSS3 Features:

  • Flexbox: A layout model that allows items to align and distribute space within a container efficiently.
  • Grid Layout: Provides a two-dimensional grid-based layout system, optimized for building complex web layouts.
  • Media Queries: Enables the creation of responsive designs by applying styles based on device characteristics like screen size and resolution.
  • Transitions: Allows for smooth changes between CSS property values over a specified duration.
  • Animations: Provides the ability to animate changes to CSS properties over time, offering more dynamic visual effects.
  • Custom Properties (Variables): Allows the definition of custom CSS variables for reuse throughout a stylesheet.
  • Transform: Enables the manipulation of elements through rotation, scaling, translation, and skewing.
  • Filter: Applies graphical effects like blur, brightness, contrast, and more to elements.
